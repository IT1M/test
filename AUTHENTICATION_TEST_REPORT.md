### 🔧 **المكونات المُختبرة:**

#### **APIs المصادقة:**
- `POST /api/auth/login` - تسجيل الدخول
- `POST /api/auth/logout` - تسجيل الخروج
- `GET /api/inventory` - API محمي للاختبار

#### **صفحات واجهة المستخدم:**
- `/en/login` - صفحة تسجيل الدخول الإنجليزية
- `/ar/login` - صفحة تسجيل الدخول العربية
- `/en/dashboard` - لوحة التحكم
- `/en/data-entry` - إدخال البيانات
- `/en/analytics` - التحليلات
- `/en/reports` - التقارير
- `/en/audit` - سجل التدقيق
- `/en/backup` - النسخ الاحتياطي
- `/en/settings` - الإعدادات
